# Titanic Data Cleaning and Visualization Project

## Overview

This project focuses on cleaning, preprocessing, and visualizing the Titanic dataset using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Tasks Performed

* Loaded and explored the dataset
* Handled missing values
* Checked for duplicate records
* Identified outliers
* Created visualizations to analyze passenger data

## Visualizations

* Survival Distribution
* Gender Distribution
* Passenger Class Distribution
* Age Distribution
* Fare Distribution
* Survival by Gender

## Conclusion

This project demonstrates basic data cleaning, preprocessing, and data visualization techniques to extract meaningful insights from a real-world dataset.
